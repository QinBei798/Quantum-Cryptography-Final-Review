# 第5章 量子随机数发生器 (QRNG)

## 5.1 随机数的定义与分级（填空/选择必考）

### 5.1.1 随机数序列的三大核心要求
1. **均匀分布 (Uniformity)**：数列中每个数出现的频率应相等或近似相等。$p(r) = \frac{1}{n}$。
2. **独立性 (Independence)**：数列中任意一个数都不能由其他数推导出来。满足联合概率拆分：$p(r_1 r_2) = p(r_1)p(r_2)$。
3. **不可预测性 (Unpredictability)**：给定过去的序列或外部环境攻击信息 $\lambda$，无法预测下一位的值。$p(r|\lambda) = p(r)$。
> **考点结论**：真随机数列天然满足独立性，因此是严格**不可预测**的。

### 5.1.2 随机数发生器三大分类对比

| 发生器类型 | 核心物理/数学原理 | 特点与局限性 | 安全性级别 |
| :--- | :--- | :--- | :--- |
| **[[伪随机数发生器|伪随机数发生器 (PRNG)]]** | 基于确定性计算机算法（如线性同余、混沌映射），依赖经典**种子 (Seed)** 展开。 | 周期性存在、一旦种子泄露或被穷举，后续序列**完全可预测**。 | 弱（不可用于高安全密码学） |
| **[[真随机数发生器|物理真随机数发生器 (TRNG)]]** | 基于经典宏观物理热噪声、电子震荡、流体混沌。 | **确定性系统**。理论上如果把所有微观分子运动、环境参数全部精确测量（即掌握了算力极高的隐藏变量 $\lambda$），它就是可预测的。 | 中 |
| **[[量子随机数发生器|量子随机数发生器 (QRNG)]]** | 基于微观量子效应（如单光子路径选择、真空场涨落）。 | 核心基于量子力学的**内在随机性**（塌缩本质随机），具有**原理上的不可预测性**。 | 最高（信息论安全） |

## 5.2 QRNG 的三大核心物理实现方案（高频简答题）

实际中，根据所利用的量子物理现象不同，QRNG 主要分为以下三类：

### 1. 基于单光子探测的 QRNG（空间/时间特性）
- **[[光子分束器态选择]]**（最经典直观模型）：
  - **原理**：将一个单光子投射到 $50:50$ 的光学分束器（BS）上。单光子处于透过与反射的叠加态：$|\psi\rangle = \frac{1}{\sqrt{2}}(|t\rangle + |r\rangle)$。
  - **编码**：如果在透射方向探测器 $D_0$ 响应，编码为 `0`；在反射方向探测器 $D_1$ 响应，编码为 `1`。每次塌缩完全随机。
- **单光子到达时间发生器**：
  - **原理**：利用连续单光子到达探测器的时间间隔（时钟周期数的奇偶性）来编码随机数。



### 2. 基于连续变量的量子场涨落 QRNG（高速主流方案）
- **[[真空态涨落 QRNG]]**：
  - **原理**：量子力学中，真空并不空，存在不确定性原理限制的**零点能场强涨落**（真空噪声）。
  - **实现**：用一束强本振激光与真空态在分束器上混合，通过平衡零差探测器（BHD）放大并测量其正交振幅或正交相位涨落。将测得的连续高斯分布波形通过模数转换（ADC）截取成二进制01。由于突破了单光子计数的死时间限制，速度极大提升。

### 3. 基于激光器自发辐射相位噪声的 QRNG
- **原理**：激光器在刚启振或处于非线性状态时，自发辐射产生的微观**相位噪声**是完全随机的。通过干涉仪将相位涨落转化为强度涨落进行探测。

## 5.3 随机性后处理与纠偏算法（大题计算必考！）

> 现实痛点：由于物理器件的不完美（如分束器不是严格的 $50:50$，或者探测器有暗计数差异），原始采集的 01 序列往往伴随有**偏置（Bias）**（即 `0` 和 `1` 的概率不完全相等，$p \neq 0.5$）。必须通过后处理算法进行**随机性提取**。

### 5.3.1 [[Von Neumann 纠偏算法]]（经典计算题）
- **核心逻辑**：将原始序列**两两不重叠地分组成对**，按照以下映射规则处理：

| 连续两位原始输入 | 后处理输出结果 | 发生概率 |
| :---: | :---: | :---: |
| `00` | **丢弃**，不输出 | $p^2$ |
| `11` | **丢弃**，不输出 | $(1-p)^2$ |
| `01` | 输出 **`0`** | $p(1-p)$ |
| `10` | 输出 **`1`** | $(1-p)p$ |

- **数学证明（为什么能消偏）**：
  因为原始出现 `01` 的概率为 $p(1-p)$，出现 `10` 的概率为 $(1-p)p$。由乘法交换律可知：
  $$P(\text{输出}0) = P(\text{输出}1) = p(1-p)$$
  因此，无论原始偏置 $p$ 有多严重（只要 $p \neq 0, 1$），输出的 `0` 和 `1` 概率绝对**严格相等（50%对50%）**。
- **致命缺点**：效率极低。数据吞吐量至少被斩断大半，且产出率不稳定（取决于输入序列）。

### 5.3.2 异或（XOR）后处理
- **原理**：将多路独立且偏置相同的序列进行逐位异或。
- **数学结论**：若单路输出 0 的概率为 $p$，则 $n$ 路独立序列异或后，输出 0 的概率收敛为 $p_n(0) = \frac{1}{2}[1 + (2p-1)^n]$。当 $n$ 增大时，概率迅速向 $\frac{1}{2}$ 逼近。

### 5.3.3 信息论安全的随机性提取器 (Extractors)
当面对复杂的、带有潜在经典或量子敌手攻击相关性的非完美熵源时，必须使用强提取器。
- **[[Toeplitz提取器]]**：
  - 特点：常应用在 QKD 的保密放大步骤中。
  - 优化：利用**快速傅里叶变换 (FFT)** 可以将矩阵相乘的运行时间降低至 $\mathcal{O}(n \log n)$，适合高速 QRNG 实时处理。
- **[[Trevisan提取器]]**：
  - 特点：已被严格证明**在量子敌手攻击下仍是安全的**（Quantum-proof）。
  - 消耗：所需的输入随机种子（Seed）长度非常短，仅为初始数据长度的对数多项式函数。
