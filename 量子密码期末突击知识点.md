# 核心摘要

> 本笔记为【量子密码】期末考试极速突击版。核心考点涵盖 [[量子力学基础]]（叠加、纠缠、测量）、[[量子密钥分配]]（BB84、B92、E91）、高级安全模型（[[DI-QKD|设备无关QKD]]、[[MDI-QKD|测量设备无关QKD]]）以及 [[QSDC|量子安全直接通信]]。重点在于理解“区分非正交态必然会干扰至少其中一个态”的安全本质。

# 第1章 量子力学基础知识

## 1.1 数学基础与狄拉克符号
- **[[Hilbert空间]]**：定义了内积的复向量空间。
- **狄拉克符号 (Bra-Ket)**：
  - **右矢 (Ket)**：列单位向量，记作 $|\psi\rangle = \begin{pmatrix} a \\ b \end{pmatrix}$，表示量子系统的状态。满足归一化条件：$|a|^2 + |b|^2 = 1$。
  - **左矢 (Bra)**：行单位向量，为右矢的共轭转置，记作 $\langle\psi| = (|\psi\rangle^T)^* = \begin{pmatrix} a^* & b^* \end{pmatrix}$。
- **[[态叠加原理]]**：若系统可处于 $| \varphi \rangle$ 和 $| \psi \rangle$，则也必然可处于复合叠加态 $|\Phi\rangle = c_1|\varphi\rangle + c_2|\psi\rangle$。
- **复合量子系统**：由多个子系统复合而成，状态通过 **[[张量积]]** $\otimes$ 结合。$n$ 个量子比特系统构成 $2^n$ 维的 Hilbert 空间，存储能力呈指数增长。

## 1.2 状态基与量子比特 (Qubit)
- **单比特二维基**：
  - **$Z$ 基（计算基）**：$\{|0\rangle, |1\rangle\}$，其中 $|0\rangle = \begin{pmatrix} 1 \\ 0 \end{pmatrix}$, $|1\rangle = \begin{pmatrix} 0 \\ 1 \end{pmatrix}$。
  - **$X$ 基（对角基）**：$\{|+\rangle, |-\rangle\}$，其中 $|+\rangle = \frac{1}{\sqrt{2}}(|0\rangle+|1\rangle)$, $|-\rangle = \frac{1}{\sqrt{2}}(|0\rangle-|1\rangle)$。
- **基间转换公式**（**考试计算必考**）：
  - $|+\rangle = \frac{1}{\sqrt{2}}(|0\rangle+|1\rangle)$ ， $|-\rangle = \frac{1}{\sqrt{2}}(|0\rangle-|1\rangle)$
  - $|0\rangle = \frac{1}{\sqrt{2}}(|+\rangle+|-\rangle)$ ， $|1\rangle = \frac{1}{\sqrt{2}}(|+\rangle-|-\rangle)$

## 1.3 量子操作与逻辑门
> 所有孤立系统的演化和逻辑操作在数学上都对应一个 **[[么正矩阵|么正算子(酉算子) U]]**，满足 $U^\dagger = U^{-1}$，演化具有可逆性和并行性。

- **常见单比特操作（[[泡利矩阵|Pauli算子]]）**：
  - $I = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}$ （不操作）
  - $X = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}$ （取反门/比特翻转）：$X|0\rangle=|1\rangle, X|1\rangle=|0\rangle$
  - $Z = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$ （相位翻转门）：$Z|0\rangle=|0\rangle, Z|1\rangle=-|1\rangle$
  - $Y = \begin{pmatrix} 0 & -i \\ i & 0 \end{pmatrix}$ ：$Y|0\rangle=i|1\rangle, Y|1\rangle=-i|0\rangle$
  - **[[Hadamard门|H门]]**：$H = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}$，用于制备均匀叠加态：$H|0\rangle = |+\rangle, H|1\rangle = |-\rangle$。
- **常见两位门（[[CNOT门|控制非门]]）**：
  - 作用：若控制位为 $|1\rangle$ 则翻转目标位，否则保持。
  - 映射关系：$|00\rangle\rightarrow|00\rangle, |01\rangle\rightarrow|01\rangle, |10\rangle\rightarrow|11\rangle, |11\rangle\rightarrow|10\rangle$。

## 1.4 量子测量与三大物理原理
- **[[投影测量]]**：用一组完备正交基 $\{|u_i\rangle\}$ 测量态 $|\phi\rangle = \sum c_i|u_i\rangle$，测得结果 $|u_i\rangle$ 的概率为 **$|c_i|^2$**，测量后状态立即 **塌缩** 到 $|u_i\rangle$。
- **[[不可克隆定理]]**：无法完美克隆一个任意的、未知的量子态。
- **[[不确定性原理|测不准原理（辨别定理）]]**：
  - 核心本质：`非正交态不可完美区分`。
  - 严格表述：**区分非正交态必然会干扰至少其中一个态**（这是所有QKD安全的物理基石）。

## 1.5 量子纠缠与Bell态
- **[[量子纠缠]]**：相距遥远的多个粒子之间存在非定域、非经典的关联，对其中一个粒子的测量会导致联合系统瞬间塌缩。
- **四个 [[Bell态]]（EPR对，最大纠缠态）**：
  - $|\Phi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle+|11\rangle)$
  - $|\Phi^-\rangle = \frac{1}{\sqrt{2}}(|00\rangle-|11\rangle)$
  - $|\Psi^+\rangle = \frac{1}{\sqrt{2}}(|01\rangle+|10\rangle)$
  - $|\Psi^-\rangle = \frac{1}{\sqrt{2}}(|01\rangle-|10\rangle)$
- **[[超密编码]]**：利用1个纠缠对，Alice仅对自身粒子进行局域 Pauli 操作（$I, Z, X, iY$），即可向Bob传输 **2 bit 经典信息**。
- **[[量子隐形传态|量子隐形传态 (Teleportation)]]**：利用共享纠缠与经典信道，将未知的单粒子量子态传输给Bob，而粒子本身不发生空间移动（非超光速，不违背不可克隆）。

# 第2章 量子密钥分配 (QKD) 经典协议

## 2.1 [[BB84协议]]（重点！）
### 2.1.1 协议执行步骤
1. **信息传输**：Alice 随机选择比特值（0或1）与调制基（$Z$ 基或 $X$ 基），制备对应的单光子状态发给 Bob。
2. **基比对**：Bob 随机选择 $Z$ 基或 $X$ 基进行测量。随后双方通过经典信道公开**各自使用的测量基**。
3. **保留生密钥**：双方只保留`测量基相同的比特`，形成原始生密钥。
4. **后处理**：
   - **[[窃听检测]]**：随机抽取部分比特比对错误率。若错误率高于**阈值**，说明存在窃听，放弃密钥；
   - **[[纠错]]**：消除信道噪声导致的比特不一致；
   - **[[保密增强]]**：通过矩阵压缩密钥长度，将窃听者 Eve 可能获得的信息压缩至任意小。

### 2.1.2 经典攻击：[[截获-重发攻击]]
- Eve 截获 Alice 的光子，随机选基测量，并根据测量结果制备新光子发给 Bob。
- **数学结论**：Eve 选错基的概率为 $50\%$。在选错基的情况下，Bob 选对基却测出错误比特的概率为 $50\%$。因此，截获-重发攻击将不可避免地在最终留下的生密钥中引入 **$25\%$ 的固有错误率**。

### 2.1.3 安全性阈值
- **无条件安全界限**：安全密钥量上限为 $I(A,B) - I(A,E)$。
- **理论硬指标**：利用纠缠提纯证明，当极化信道的随机比特错误率 **$e \le 11\%$** 时，协议是绝对保密的；高于 $11\%$ 必须放弃密钥。

## 2.2 其他QKD经典协议
- **[[B92协议]]**：Alice 仅使用两个非正交态（如 $|0\rangle$ 和 $|+\rangle$）分别代表0和1。Bob 用共轭基测量，若测得 $|1\rangle$ 或 $|-\rangle$ 则为确定性有效位。
- **[[E91协议]]**：基于纠缠对的 QKD。Alice 和 Bob 随机选三种方向测量。利用**相同基**的测量结果生成密钥（结果完全相反），利用**不同基**的测量结果通过验证 **[[Bell不等式]]**（CHSH值超经典界 $2$，最大达 $2\sqrt{2}$）来检测窃听。

# 第3章 理论进阶：正交态协议与量子并行计算

## 3.1 两类QKD协议的本质：[[信息分割]]
- 误解：“量子密码只能基于非正交态”。
- **[[正交态协议|正交态QKD协议（如GV95、LL02）]]**：
  - 原理：将单个光子分成两个不同的波包在不同时间发送，或将 Bell 态的两个粒子分步发送。
  - 本质结论：两类协议并不对立。其核心都是 **信息分割**（防止敌手同时拿到两部分量子或经典信息）。在信道中，`分割后传输的载体态本质上依然是非正交或最大混合态`，Eve 盲目测量必引入干扰。

## 3.2 量子并行计算与经典密码的威胁
- **原理**：利用 $H^{\otimes n}$ 作用于 $|0\otimes n\rangle$ 产生全空间均匀叠加态 $\frac{1}{\sqrt{N}}\sum|x\rangle$，通过单次酉操作 $U_f$ 即可同时计算所有 $x$ 的函数值 $|x\rangle|f(x)\rangle$。
- **[[Shor算法]]**（公钥密码体制的死敌）：
  - 核心：利用量子傅里叶变换在多项式时间 $O((\log N)^3)$ 内求解大数在模 $N$ 下的阶（周期）。
  - **被完全攻破的体制**：[[RSA]]（大数分解）、[[DH密码|DH]] / [[ElGamal]]（离散对数）、[[ECC]]（椭圆曲线密码）。
- **[[Grover算法]]**（对称密码体制的威胁）：
  - 核心：量子无序数据库搜索，提供**平方加速**（经典 $O(N/M)$，量子 $O(\sqrt{N/M})$）。
  - **影响**：使得对称密码（如 [[AES]]、[[DES]]）和 Hash 算法的**有效密钥长度直接缩短一半**（应对策略：密钥长度翻倍，如启用 AES-256）。
- **[[Simon算法]]**：快速求解二对一周期函数的模二加周期。实现指数加速（经典 $O(2^{n/2})$，量子 $O(n)$）。可用于攻破多种密码结构及对称体制下的 [[MAC|消息认证码]]（存在性伪造）。

# 第4章 实用化前沿：DI-QKD、MDI-QKD 与 QSDC

## 4.1 设备安全性漏洞与对抗方案
> 实际系统中，光源的不完美（多光子发射）会引发 **[[PNS攻击|PNS(光子数分离)攻击]]**；单光子探测器由于非理想性（暗计数、死时间等），极易受到侧信道攻击（如探测器控制攻击、探测器盲化攻击）。

- **[[诱骗态方法]]**（Decoy State）：Alice 随机发送不同强度的弱相干脉冲（信号脉冲 + 诱骗脉冲）。通过比对不同强度的探测响应率，完全规避针对非理想光源的 PNS 攻击，将成码率提高至与信道损耗呈线性关系 $O(\eta)$。
- **[[DI-QKD|设备无关QKD (Device-Independent QKD)]]**：
  - 原理：完全不依赖对量子设备的信任，将输入输出视为黑盒，通过产生**最大 Bell 不等式（CHSH）违背**来证明安全性。
  - 缺点：对探测效率要求极端严苛（接近1），成码率极低，目前无法实用。
- **[[MDI-QKD|测量设备无关QKD (Measurement-Device-Independent QKD)]]**（**高频考点**）：
  - 架构：Alice 和 Bob 随机制备微弱量子态发给不合法的第三方（中继 Charlie），Charlie 仅做 **Bell基测量** 并公布结果。
  - 安全本质：把最容易被黑客侧信道攻击的“测量设备”直接交给窃听者，窃听者依然无法获知密钥。`彻底免疫了所有针对测量设备的侧信道攻击`。实用性极强。
- **[[TF-QKD|双场QKD (Twin-Field QKD)]]**：最新进展，两端直接用相干光干涉，成功克服无中继 QKD 的率距限制，使安全成码率达到开方关系 **$O(\sqrt{\eta})$**，无中继传输距离已突破 1000 公里。

## 4.2 量子安全直接通信 (QSDC)
- **[[QSDC|QSDC (Quantum Secure Direct Communication)]]**：
  - 区别点：QKD 只生成随机密钥，不传明文；QSDC **直接安全地传输秘密消息本身**。
  - 模型特征：采用双向量子信道（如 Ping-Pong 协议）。信息编码在 Alice 所采取的 **局域量子操作** 中，而不在具体的载体状态中。
  - 模式划分：包含 **消息模式**（传密文）和 **控制模式**（检测窃听）。
  - 局限性：易遭受拒绝服务（[[DOS攻击|DOS]]）干扰攻击；安全性属于“准安全”（一旦泄露不可撤回，需要依赖后处理或经典认证机制辅助）。
