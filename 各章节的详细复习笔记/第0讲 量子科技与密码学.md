# 核心摘要

> 本章为【量子密码学】的全局总览。核心考点在于理清**两次量子革命的演进**、**量子三大核心物理特性**（叠加、纠缠、测量）、**四大典型量子算法对传统密码的理论威胁**（Shor、Simon、Grover、BHT），以及**量子密码（QKD）与后量子密码（PQC）的本质区别与共存关系**。必须牢记“量子密码的核心意义是极大拉高‘密钥分配’这条木板的高度”。

# 第0章 量子科技与密码学

## 0.1 两次量子力学革命
- [cite_start]**第一次量子科技革命**（上世纪40年代兴起）：催生了原子弹、半导体晶体管、激光器等重要成果 [cite: 2]。
- [cite_start]**第二次量子科技革命**（上世纪末以来）：在信息技术领域兴起，以调控量子状态进行信息处理为主，如今已接近产业化阶段 [cite: 2]。
  - **标志性历史突破**：
    - [cite_start]1900年：普朗克建立 **[[黑体辐射定律]]**，诞生普朗克常数 $h$ [cite: 2]。
    - [cite_start]1905年：爱因斯坦解释 **[[光电效应]]**，发现光量子（光子） [cite: 2]。
    - [cite_start]1913年：玻尔提出氢原子 **[[分立能级]]** 理论 [cite: 3, 4]。
    - [cite_start]1923年：德·布罗意提出微观粒子具有 **[[波粒二象性]]** [cite: 4]。
    - [cite_start]1926年：薛定谔提出 **[[薛定谔波动方程]]**（波动力学） [cite: 4][cite_start]；海森堡等人建立 **[[矩阵力学]]**，两者数学形式完全等价 [cite: 5]。

## 0.2 量子密码的核心物理基础
- [cite_start]**[[量子比特|量子比特 (Qubit)]]**：经典比特只能是 0 或 1，而 Qubit 能够处于不同状态的 **[[态叠加原理|叠加态]]**：$|\psi\rangle = \alpha|0\rangle + \beta|1\rangle$（其中 $|\alpha|^2 + |\beta|^2 = 1$） [cite: 4, 5]。
- [cite_start]**[[狄拉克符号]]**：右矢 $|\psi\rangle$ 表示二维列向量，左矢 $\langle\psi|$ 表示共轭行向量 [cite: 5, 6]。
- [cite_start]**[[复合量子系统]]**：多比特系统通过 **[[张量积]]** $\otimes$ 结合，存储能力随比特数 $n$ 呈 **指数级增长**（$\Sigma a_i|i\rangle$，维数为 $2^n$） [cite: 6]。
- [cite_start]**[[么正操作|量子操作（么正变换）]]**：所有局域孤立系统的演化对应酉矩阵 $U$（满足 $U^\dagger = U^{-1}$），具有天然的 **[[量子并行性]]** [cite: 6]。
- [cite_start]**[[投影测量]]**：测量会将状态随机塌缩到基向量上，测得状态 $|u_i\rangle$ 的概率为对应系数的模平方 $|c_i|^2$ [cite: 7, 8]。
- **三大核心防御防线**：
  - [cite_start]**[[不可克隆定理]]**：未知的单量子态绝对不可被完美复制 [cite: 9]。
  - [cite_start]**[[测不准原理|不确定性原理]]**：未知量子态不可准确测量，且测量往往会改变量子态本身 [cite: 9]。
  - [cite_start]**[[量子纠缠]]**：相距遥远的粒子间存在非定域、非经典的强烈关联。对 A 粒子的测量会导致联合系统瞬时塌缩，从而决定 B 粒子的状态（2022年诺贝尔物理学奖通过 Bell 不等式违背实验彻底证实） [cite: 9, 13]。

## 0.3 量子计算对现行密码体制的威胁
> [cite_start]现行体制普遍采用**混合密码体制**：公钥密码用于分发会话密钥，对称密码用于加密大批量数据 [cite: 82]。

### 0.3.1 公钥密码的覆灭：[[Shor算法]]
- [cite_start]**原理**：利用量子傅里叶变换，在多项式时间 $\mathcal{O}((\log N)^3)$ 内求解周期，实现对传统数学难题的 **指数级加速** [cite: 23, 24]。
- [cite_start]**完全攻破的体制**：[[RSA]]（大数分解）、[[DH密码|DH / ElGamal]]（离散对数）、[[ECC]]（椭圆曲线密码） [cite: 24, 82]。
- [cite_start]**补充前沿（变分量子算法 VQA）**：类似神经网络的启发式算法，适用于近期的 NISQ 设备，已仿真实现了对大数分解及格密码（SVP/LWE问题）的攻击尝试 [cite: 46, 48, 49]。

### 0.3.2 对称密码与哈希体系的削弱
- [cite_start]**[[Grover算法]]**：提供 **平方级加速**（经典 $\mathcal{O}(N)$，量子 $\mathcal{O}(\sqrt{N})$），专门用于穷举搜索 [cite: 50, 82][cite_start]。使得 AES/DES 的 **有效密钥长度直接减半**（应对策略：密钥长度翻倍，如改用 AES-256） [cite: 50, 82]。
- [cite_start]**[[Simon算法]]**：快速寻找二对一函数周期，实现 **指数级加速**（经典 $\mathcal{O}(2^{n/2})$，量子 $\mathcal{O}(n)$），能对多种对称密码结构及 [[MAC|消息认证码]] 实施区分攻击与“存在性伪造” [cite: 56, 58, 81]。
- [cite_start]**[[BHT算法]]**：用于 **[[量子碰撞搜索]]**，可将哈希冲突的寻找复杂度降至 $\mathcal{O}(\sqrt[3]{N})$，直接威胁 [[SHA2]]、[[SHA3]] 等哈希函数的安全边界 [cite: 69, 70, 72]。

## 0.4 抗量子计算攻击的防守双璧
[cite_start]为了对抗 Shor 和 Grover 算法的严重威胁，目前密码学界演化出两条截然不同的技术路线 [cite: 84]：

1. **[[量子密码|量子密码 (Quantum Cryptography / QKD)]]**：
   - [cite_start]**原理**：利用单光子的不可克隆与非正交测量干扰等 **物理学定律** 来保证安全 [cite: 85, 117]。
   - [cite_start]**特点**：提供 **信息论安全（无条件安全）** 的密钥分配，与“一次一密 (OTP)”结合能达到绝对保密通信 [cite: 84, 116]。
2. **[[后量子密码|后量子密码 (Post-Quantum Cryptography / PQC)]]**：
   - [cite_start]**原理**：完全运行在经典计算机上，但基于当前已知量子算法无法攻破的 **经典数学难题**（如格密码、多变量密码、基于哈希的签名等） [cite: 116, 120]。
   - [cite_start]**特点**：灵活易用、部署成本低，是当前量子密码的最强有力竞争对手（NIST 正在推行其算法标准化） [cite: 120]。

## 0.5 误区澄清与系统安全性（期末辨析易错点！）
- **❌误区 1：量子密码天生绝对安全**
  - [cite_start]**正解**：理论上达到信息论安全。但在实际工业落地中，系统常因器件不完美而面临 **[[侧信道攻击]]**（如光源、探测器漏洞），绝对安全不存在 [cite: 116]。
- **❌误区 2：利用量子纠缠可以实现超光速通信**
  - [cite_start]**正解**：纠缠态的塌缩虽然是瞬时的，但这**并不能传递确定性的确定消息**（无法传递经典信息）。在 [[量子隐形传态]] 中，量子态的完全转移必须依赖经典信道协同，速度绝不会超越光速 [cite: 116, 117]。
- **❌误区 3：有了量子密码，经典密码就可以被彻底淘汰了**
  - [cite_start]**正解**：目前狭义的量子通信通常指 **[[QKD|量子密钥分发]]**，它**不是新的通信方式，也不能做新加密，只能解决高安全性的“密钥分配”问题**。其余的大部分密码学功能（如密钥管理、存储、防病毒木马等）依然必须完全依赖经典安全体系 [cite: 117]。
- **💡系统安全的“木桶原则”**：
  - [cite_start]一个信息安全系统的整体安全性由其最低的一块木板决定 [cite: 118][cite_start]。量子密码的终极意义不是包治百病，而是**把“密钥分配”这块极其脆弱的短板拉到了近乎无限高**的物理安全级别 [cite: 118]。
