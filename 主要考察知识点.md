# 考点一：基础知识到隐形传态（全卷高频计算与辨析底座）

## 1. 狄拉克符号与 Hilbert 空间运算
- **右矢 (Ket)**：$|\psi\rangle = \begin{pmatrix} a \\ b \end{pmatrix}$，表示系统状态的列向量。满足归一化条件：$\langle\psi|\psi\rangle = |a|^2 + |b|^2 = 1$。
- **左矢 (Bra)**：$\langle\psi| = (|\psi\rangle^T)^* = \begin{pmatrix} a^* & b^* \end{pmatrix}$，为右矢的共轭转置。
- **内积与正交**：$\langle\varphi|\psi\rangle$ 结果为复数。若 $\langle\varphi|\psi\rangle = 0$，则两状态严格**正交**。
- **外积与算子**：$|\varphi\rangle\langle\psi|$ 结果为一个矩阵。
- **[[态叠加原理]]**：若系统可处于 $| \varphi \rangle$ 和 $| \psi \rangle$，则必可处于其线性叠加态 $c_1|\varphi\rangle + c_2|\psi\rangle$。多比特系统通过**张量积** $\otimes$ 结合，空间维数随比特数 $n$ 呈 $2^n$ **指数级增长**。

## 2. 两套标准基与基变换（大题必考核心）
- **$Z$ 基（计算基）**：$\{|0\rangle, |1\rangle\}$，其中 $|0\rangle = \begin{pmatrix} 1 \\ 0 \end{pmatrix}, |1\rangle = \begin{pmatrix} 0 \\ 1 \end{pmatrix}$。
- **$X$ 基（对角基）**：$\{|+\rangle, |-\rangle\}$，其中 $|+\rangle = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ 1 \end{pmatrix}, |-\rangle = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ -1 \end{pmatrix}$。
- **极重要基变换公式（手写大题必备）**：
  - $|+\rangle = \frac{1}{\sqrt{2}}(|0\rangle+|1\rangle)$ ， $|-\rangle = \frac{1}{\sqrt{2}}(|0\rangle-|1\rangle)$
  - $|0\rangle = \frac{1}{\sqrt{2}}(|+\rangle+|-\rangle)$ ， $|1\rangle = \frac{1}{\sqrt{2}}(|+\rangle-|-\rangle)$

## 3. 么正操作（酉算子）与四大泡利门
系统物理演化由么正算子 $U$ 描述，满足 $U^\dagger U = I$（保内积、可逆、概率总和守恒）。
- $X = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}$ （取反门）：$X|0\rangle=|1\rangle, X|1\rangle=|0\rangle$
- $Z = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$ （相位翻转门）：$Z|0\rangle=|0\rangle, Z|1\rangle=-|1\rangle$
- $Y = \begin{pmatrix} 0 & -i \\ i & 0 \end{pmatrix}$ ：$Y|0\rangle=i|1\rangle, Y|1\rangle=-i|0\rangle$
- **[[Hadamard门|H门]]**：$H = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}$，制备均匀叠加态：$H|0\rangle = |+\rangle, H|1\rangle = |-\rangle$。

## 4. 投影测量与三大物理定理
- **[[投影测量]]**：测得结果 $|u_i\rangle$ 的概率为对应分量的模平方 **$|c_i|^2$**，测量后状态立即**塌缩**到 $|u_i\rangle$。
- **[[不可克隆定理]]**：未知单量子态绝对不可完美复制。
- **[[不确定性原理|测不准原理的误区澄清]]**：用海森堡测不准原理（$\Delta(Z)\Delta(X)\ge 0$）解释 BB84 非正交态不可同时测准是**不恰当的**。BB84 安全性的物理基石是 **“非正交态不可区分定理”**：**区分非正交态必然会干扰至少其中一个态**。

## 5. 量子纠缠与[[量子隐形传态]]
- **四个 [[Bell态]]（最大纠缠态）**：
  - $|\Phi^\pm\rangle = \frac{1}{\sqrt{2}}(|00\rangle \pm |11\rangle)$ （同向同性关联）
  - $|\Psi^\pm\rangle = \frac{1}{\sqrt{2}}(|01\rangle \pm |10\rangle)$ （异向同性关联）
- **[[量子隐形传态|量子隐形传态 (Teleportation)]]**：利用共享纠缠与经典信道，将未知的单粒子量子态传输给 Bob。
  - **辨析踩分点**：不是“瞬间”转移，完全转移所需时间**不低于光速**（因为 Bob 必须等待 Alice 通过经典信道传来测量结果才能做对应的 Pauli 修正操作）；目前只能传量子态，不能传宏观物质。

# 考点二：QKD BB84 协议（全卷绝对大题重灾区）

## 1. 协议核心四大步骤
1. **制备与发送**：Alice 随机选择比特值（0或1）与调制基（$Z$ 或 $X$ 基），制备单光子发给 Bob。
2. **随机测量**：Bob 在不知情下随机选 $Z$ 或 $X$ 基测量。
3. **基比对 (Sifting)**：双方通过经典信道公开测量基，**仅保留测量基相同的比特轮次**，形成 **原始生密钥 (Raw Key)**。
4. **后处理**：
   - **[[窃听检测]]**：抽取部分比特比对误码率，若高于**安全阈值（理论硬指标约为 11%）**则放弃密钥；
   - **[[纠错|原始纠错]]** 与 **[[保密增强]]**（利用保密放大/压缩矩阵消除 Eve 的残余信息）。

## 2. 截获-重发攻击 (Intercept-Resend) 25% 错误率严格证明

- **证明推导（简答/大题核心）**：
  1. 假设 Alice 与 Bob 在生密钥轮次均选择 $Z$ 基，Alice 发射 $|0\rangle$。
  2. 敌手 Eve 拦截，由于不知道基，她有 **50% 概率选错基**（选了 $X$ 基）。
  3. 当 Eve 选错基用 $X$ 基测 $|0\rangle$ 时，状态立即塌缩为 $|+\rangle$ 或 $|-\rangle$（概率各50%）。Eve 只能被迫将塌缩后的伪造态（如 $|+\rangle$）重发给 Bob。
  4. Bob 用原本正确的 $Z$ 基去测这个受污染的 $|+\rangle$。由于 $|+\rangle = \frac{1}{\sqrt{2}}(|0\rangle+|1\rangle)$，Bob 测得错误比特 `1` 的概率为 **50%**。
  5. **总误码率统计**：
     $$\text{总错误率} = P(\text{Eve选错基}) \times P(\text{Bob在错态下测错}) = 50\% \times 50\% = 25\%$$

# 考点三：QSDC（量子安全直接通信）四个要求与机制

## 1. QSDC 核心“四个要求”（大纲明确划定，必背简答）
根据课件对安全直接通信的刚性定义，QSDC 系统必须严格满足以下四个核心安全与功能边界：
1. **直接传输性**：不依赖事先建立的随机密钥和加解密中间缓冲，**直接安全地传输秘密消息本身**。
2. **窃听检测优先（先检测后释放原则）**：在秘密信息被完全解码前，必须能够完成信道窃听检测。
3. **秘密信息的“准安全”伪装（最大混合态）**：秘密信息在信道中传输时，对任何敌手而言其整体状态必须表现为无任何非定域关联的**最大混合态**，Eve 盲目测量必引入可察觉的干扰。
4. **后处理的可控制性**：必须具备完备的控制模式和纠错后处理机制，确保在噪声或攻击超标时能即时强制终止，防止明文泄露扩大。

## 2. 典型协议：Ping-Pong 协议
- **机制**：基于纠缠的双向量子信道。Bob 制备 Bell 态 $|\Psi^+\rangle$，将信号粒子 1 发给 Alice（Ping阶段）。
- **Alice 的双模式博弈**：
  - **[[控制模式|控制模式 (C-mode)]]**：随机抽取粒子用 $Z$ 或 $X$ 基测量并公开，Bob 测家庭粒子 2，利用**异向同性**检测窃听。
  - **[[消息模式|消息模式 (M-mode)]]**：直接对粒子 1 施加局域么正变换编码。传 `0` 作用 $I$（保持 $|\Psi^+\rangle$）；传 `1` 作用 $Z$（变为 $|\Psi^-\rangle$）。再将粒子发回（Pong阶段）。Bob 通过联合 **Bell基测量** 确定明文。

## 3. 防御关键：块传输 (Block Transmission)
- **DL04等协议的核心**：Alice 将光子打包成一个“量子状态块”整体发送。在整个块安全到达前，**绝不透露任何基底和控制光子位置**。强行逼迫截获者 Eve 直面非正交态不可区分死线，使其不敢贸然动手，从而完美解耦传输与检测的时间差。

# 考点四：量子秘密共享 (QSS) 原理与 HBB 协议

## 1. 秘密共享门限原理
- **`(k, n)` 门限方案**：分发者把消息加密分割发给 $n$ 个代理人。任意 $\ge k$ 个人联合可以完美解密，而 $< k$ 个人绝对得不到任何一点关于秘密的线索。QSS 中最常见的是 **`(n, n)` 门限方案**。
- **前置硬假设**：系统默认假设**至少有一个代理人是严格诚实的**。

## 2. HBB 协议完整过程（三粒子 GHZ 态）

1. **状态共享**：Alice、Bob 和 Charlie 三方共享一组最大纠缠三粒子态：
   $$|\psi\rangle_{ABC} = \frac{1}{\sqrt{2}}(|000\rangle + |111\rangle)_{ABC}$$
2. **局域测量**：三人分别独立、真随机地选择 **$X$ 基** 或 **$Y$ 基** 测量各自的粒子，记录私密结果 $a, b, c \in \{0, 1\}$。
3. **协同解密（踩分步骤）**：
   - Bob 和 Charlie 单方测得的结果完全是 50:50 的极大混合态，不包含任何秘密。
   - 两人通过经典信道相互告知对方自己的测量基与结果，联合算出 **$b \oplus c$**。
   - 结合 Alice 随后公开的测量基体系，通过既定的 GHZ 算子特征值本征对应表（如 $X X X$ 或 $X Y Y$ 的奇偶校验关系），两人合作即可完美反推出 Alice 的测量序列 $a$，从而以此序列作为掩码，安全还原 Alice 共享的经典核心秘密。

## 3. 安全性：不诚实内部代理人（内鬼）攻击分析
- **假 Bell 态替换攻击（ZM协议等）**：秘密共享的主要安全威胁来自内部。内鬼（如 Bob）可以通过在接收/转发阶段拦截粒子，偷偷塞入自己制备的**伪造 Bell 态对**发给后续诚实代理人，自己在后处理阶段通过联合 Bell 基测量，**绕过门限限制，在不需要其他代理人帮助下独自窃取 Alice 的全部秘密**。
- **防御**：必须在粒子流中无规律地交叉混入大量**控制模式 (C-mode)** 光子进行奇偶本征值随机抽检。

# 考点五：QRNG（量子随机数发生器）

## 1. 随机数三大特性与发生器分级
- **三大特性**：**均匀分布**（频率严格相等）、**独立性**（联合概率拆分 $p(r_1 r_2)=p(r_1)p(r_2)$）、**不可预测性**。
- **分级对比（填空高频）**：
  - `PRNG (伪随机数)`：基于确定性算法和种子，一旦种子泄露，完全可预测。
  - `TRNG (经典物理真随机数)`：基于热噪声等宏观混沌。系统本质上是确定性的，算力及隐藏变量无限大时理论上可预测。
  - `QRNG (量子随机数)`：基于微观量子塌缩的**内在随机性**，具有物理原理上的**绝对不可预测性（信息论安全）**。

## 2. 三大核心物理实现架构
1. **单光子路径态选择**：单光子打在 50:50 的**光学分束器 (BS)** 上，处于透射与反射的叠加态 $\frac{1}{\sqrt{2}}(|t\rangle+|r\rangle)$，根据两端探测器响应编码 0 或 1。
2. **真空态场涨落（主流高速）**：利用不确定性原理限制的零点能场强噪声，将真空态与强本振激光混合，通过平衡零差探测器 (BHD) 放大并采集成高斯分布的二进制随机数。
3. **激光器自发辐射相位噪声**：利用激光器启振时的微观自发辐射相位随机涨落，通过干涉仪转化为强度涨落探测。

## 3. Von Neumann 纠偏算法（大题必备计算）
- **痛点**：物理器件不完美会导致原始 01 序列出现偏置（$p(0) \neq p(1)$）。
- **算法规则映射**：将输入序列两两成对划分：
  - 输入 `01` $\rightarrow$ 输出 **`0`** （发生概率：$p(1-p)$）
  - 输入 `10` $\rightarrow$ 输出 **`1`** （发生概率：$(1-p)p$）
  - 输入 `00` 或 `11` $\rightarrow$ **直接丢弃**
- **消偏数学本质**：因为 $p(1-p) \equiv (1-p)p$，所以无论初始偏置多严重，输出的 0 和 1 概率**严格保持 50% 对 50%**。

# 考点六：比特承诺、数字签名等了解概念（概念/名词解释防守区）

## 1. 量子比特承诺 (Bit Commitment)
- **概念阶段**：包含**承诺阶段**（Alice 发送密封包）和**打开阶段**（Alice 给钥匙验证）。
- **两大特性**：**隐蔽性**（打开前 Bob 绝对猜不出比特值）与 **约束性**（Alice 承诺后绝对无法篡改）。
- **[[Mayers-Lo-Chau定理|MLC不可能性定理（BC不可能性）]]（必考辨析）**：完美的隐蔽性和完美的约束性在非相对论量子力学中不可兼得。因为若完美隐蔽，Alice 必然可以通过**局域么正变换**任意强行改变量子态的全局塌缩方向，实现天衣无缝的单方面篡改。

## 2. 仲裁量子签名 (Arbitrated Quantum Signature)
- **数字签名三大特征**：签署文件无物理附加、通过公开算法验证、拷贝件与原件完全等同。基本要求：不可伪造、不可抵赖。
- **Zeng（曾贵华）方案架构**：Alice 与仲裁者 Trent 共享密钥 $K_A$，Bob 与 Trent 共享 $K_B$。Alice 对 Bell 态粒子执行局域变换完成签名。验证时由**可信仲裁者 Trent** 收集粒子并执行 **Bell基测量**，从而用物理纠缠关联强行锁定 Alice 的不可抵赖性与 Bob 的不可伪造性。

## 3. 量子拜占庭将军协议
- **经典死线**：3个将军（1个司令，2个参与者）博弈中只要有1个内鬼，经典分布式算法**逻辑上绝对无解**。
- **量子突破 (GBKCW协议)**：利用**四粒子纠缠单重态**，可以在只有3个将军（包含1个叛徒）的极端情况下完美达成一致性。
- **内鬼的截获-重发攻击**：不诚实将军派遣士兵强行截获并使用固定基测量信道光子。**暴露机理**：强行测量会导致多方非定域纠缠单重态**瞬间彻底崩溃消失**，诚实将军在后续交叉选基验证轮次中必会测出显著的误码率，使内部叛徒攻击彻底暴露。
