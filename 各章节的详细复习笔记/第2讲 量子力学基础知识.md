# 第2章 量子力学基础知识

## 2.1 量子态的数学表示：狄拉克符号与空间

### 2.1.1 [[Hilbert空间|Hilbert 空间基础]]
- **定义**：定义了内积的复向量空间。
- **内积性质**：
  - 正定性：$\langle\psi|\psi\rangle \ge 0$，当且仅当向量为零向量时等号成立。
  - 共轭对称性：$\langle\varphi|\psi\rangle = \langle\psi|\varphi\rangle^*$。
  - 右线性：$\langle\phi | (c_1|\psi_1\rangle + c_2|\psi_2\rangle) = c_1\langle\phi|\psi_1\rangle + c_2\langle\phi|\psi_2\rangle$。

### 2.1.2 狄拉克符号 (Bra-Ket)
- **右矢 (Ket)**：$|\psi\rangle$，表示复向量空间中的一个**列向量**。
- **左矢 (Bra)**：$\langle\psi|$，表示右矢的**共轭转置（Hermitian共轭）**，即 $\langle\psi| = (|\psi\rangle^T)^*$，是一个**行向量**。
- **内积 (Inner Product)**：$\langle\varphi|\psi\rangle$，结果为一个**复数**。若 $\langle\varphi|\psi\rangle = 0$，则称两个态**正交**。
- **外积 (Outer Product)**：$|\varphi\rangle\langle\psi|$，结果为一个**矩阵（算子）**。
- **归一化条件**：物理上合法的量子态必须满足 $\langle\psi|\psi\rangle = 1$。

## 2.2 量子比特与基变换（计算大题必考！）

### 2.2.1 单比特的二维空间基
1. **$Z$ 基（计算基 / 标准基）**：$\{|0\rangle, |1\rangle\}$
   - $|0\rangle = \begin{pmatrix} 1 \\ 0 \end{pmatrix}$, $|1\rangle = \begin{pmatrix} 0 \\ 1 \end{pmatrix}$
2. **$X$ 基（对角基 / 共轭基）**：$\{|+\rangle, |-\rangle\}$
   - $|+\rangle = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ 1 \end{pmatrix}$, $|-\rangle = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ -1 \end{pmatrix}$

### 2.2.2 基变换公式（必须熟练默写）
- **从 $Z$ 基到 $X$ 基**：
  - $|+\rangle = \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle)$
  - $|-\rangle = \frac{1}{\sqrt{2}}(|0\rangle - |1\rangle)$
- **从 $X$ 基到 $Z$ 基**：
  - $|0\rangle = \frac{1}{\sqrt{2}}(|+\rangle + |-\rangle)$
  - $|1\rangle = \frac{1}{\sqrt{2}}(|+\rangle - |-\rangle)$

## 2.3 量子操作与么正演化

### 2.3.1 [[么正算子|么正变换 (Unitary Transformation)]]
- **定义**：孤立量子系统的随时间演化由么正算子 $U$ 描述，满足：
  $$U^\dagger U = UU^\dagger = I$$
  *(其中 $U^\dagger$ 为 $U$ 的共轭转置矩阵)*
- **物理本质**：么正变换是保内积的（$\langle U\varphi|U\psi\rangle = \langle\varphi|\psi\rangle$），因此它**保证了演化前后概率总和始终为 1**，且演化过程完全**可逆**。

### 2.3.2 核心逻辑门矩阵（必须死记硬背）
- **[[泡利矩阵|Pauli 矩阵]]**：
  - $I = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}$ （恒等算子）
  - $X = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}$ （比特翻转门/非门）：$X|0\rangle = |1\rangle$, $X|1\rangle = |0\rangle$
  - $Z = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$ （相位翻转门）：$Z|0\rangle = |0\rangle$, $Z|1\rangle = -|1\rangle$
  - $Y = \begin{pmatrix} 0 & -i \\ i & 0 \end{pmatrix}$ ：$Y = iXZ$，作用效果为 $Y|0\rangle = i|1\rangle$, $Y|1\rangle = -i|0\rangle$
- **[[Hadamard门|Hadamard 门 (H 门)]]**（制备叠加态的核心）：
  - $H = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}$
  - 作用效果：$H|0\rangle = |+\rangle$, $H|1\rangle = |-\rangle$, $H|+\rangle = |0\rangle$, $H|-\rangle = |1\rangle$

## 2.4 量子测量与状态塌缩（高分计算题常客）

### 2.4.1 [[投影测量|投影测量 (Projective Measurements)]]
- 一个可观测量对应一个 Hermitian 算子 $M$（满足 $M^\dagger = M$），可谱分解为 $M = \sum_m m P_m$，其中 $m$ 行为测量结果特征值，$P_m$ 为向对应特征空间的投影算子（$P_m = |m\rangle\langle m|$）。
- **概率计算公式**：对状态 $|\psi\rangle$ 进行测量，得到结果 $m$ 的概率为：
  $$p(m) = \langle\psi|P_m|\psi\rangle = |\langle m|\psi\rangle|^2$$
- **测量后的状态塌缩**：一旦测得结果为 $m$，系统状态立即强行塌缩为：
  $$|\psi'\rangle = \frac{P_m|\psi\rangle}{\sqrt{p(m)}} = |m\rangle$$

### 2.4.2 典型考题直觉：选错基测量
- 场景：Alice 发送 $|+\rangle$，Bob 错误地使用了 $Z$ 基 $\{|0\rangle, |1\rangle\}$ 去测量。
- 计算过程：
  - 测得结果为 `0` 的概率：$p(0) = |\langle 0|+\rangle|^2 = |\frac{1}{\sqrt{2}}(\langle 0|0\rangle + \langle 0|1\rangle)|^2 = |\frac{1}{\sqrt{2}}(1 + 0)|^2 = \frac{1}{2}$。
  - 此时状态立刻塌缩为：$|0\rangle$。
  - 同理，测得结果为 `1` 的概率也是 $\frac{1}{2}$，状态塌缩为 $|1\rangle$。

## 2.5 密度算子与复合系统

### 2.5.1 [[密度算子|密度算子 (Density Operator / 密度矩阵)]]
- **引入原因**：用于描述无法用单一右矢表示的**混合态**（即以概率 $p_i$ 处于状态 $|\psi_i\rangle$ 的系综）。
- **定义式**：$\rho = \sum_i p_i |\psi_i\rangle\langle\psi_i|$。
- **纯态与混合态的判别定理**（选择/填空必考）：
  - 若 $\text{tr}(\rho^2) = 1$，则该状态为 **纯态**。
  - 若 $\text{tr}(\rho^2) < 1$，则该状态为 **混合态**。

### 2.5.2 复合系统与[[量子纠缠]]
- 两个独立子系统 $A$ 和 $B$ 联合构成的复合空间通过 **[[张量积]]** $A \otimes B$ 表示。
- **量子纠缠定义**：若复合系统的状态 $|\Delta\rangle_{AB}$ **无法**写成两个子系统各自状态的张量积形式（即 $|\Delta\rangle_{AB} \neq |\psi\rangle_A \otimes |\phi\rangle_B$），则称该状态为纠缠态。
- **四大最大纠缠态 (Bell 态 / EPR 对)**：
  - $|\Phi^\pm\rangle = \frac{1}{\sqrt{2}}(|00\rangle \pm |11\rangle)$
  - $|\Psi^\pm\rangle = \frac{1}{\sqrt{2}}(|01\rangle \pm |10\rangle)$

## 2.6 物理三大基本定理（论述简答题核心）

### 2.6.1 [[不可克隆定理|不可克隆定理 (No-Cloning Theorem)]]
- **严格数学证明简写（大题必背）**：
  假设存在一个通用的么正克隆操作 $U$，能复制未知态 $|\psi\rangle$ 和 $|\varphi\rangle$ 到初始态 $|s\rangle$ 上：
  1. $U|\psi\rangle|s\rangle = |\psi\rangle|\psi\rangle$
  2. $U|\varphi\rangle|s\rangle = |\varphi\rangle|\varphi\rangle$
  3. 计算两式内积的左边：$(\langle\psi|\langle s|U^\dagger) (U|\varphi\rangle|s\rangle) = \langle\psi|\langle s|I|\varphi\rangle|s\rangle = \langle\psi|\varphi\rangle \cdot \langle s|s\rangle = \langle\psi|\varphi\rangle$。
  4. 计算两式内积的右边：$(\langle\psi|\langle\psi|)(|\varphi\rangle|\varphi\rangle) = (\langle\psi|\varphi\rangle)^2$。
  5. 从而导出结论：$\langle\psi|\varphi\rangle = (\langle\psi|\varphi\rangle)^2$。
  6. **矛盾点**：该方程只有在 $\langle\psi|\varphi\rangle = 0$（彼此正交）或 $\langle\psi|\varphi\rangle = 1$（完全相同）时才成立。因此，**完美的通用克隆机物理上绝对不存在**。

### 2.6.2 [[不确定性原理|测不准原理的常见误区澄清（核心考点）]]
- **课件强调**：教科书里常见的海森堡测不准关系式 $\Delta C \cdot \Delta D \ge \frac{1}{2}|\langle\psi|[C,D]|\psi\rangle|$，在状态为 $|0\rangle$、算子为 $Z$ 和 $X$ 时，右边算出来是 $0$。也就是说，$\Delta(Z)\Delta(X) \ge 0$ **并不能**在数学上推导出“其中至少一个测不准”。
- **标准判卷表述**：因此，用“测不准原理”来解释 BB84 协议中非正交态不可同时测准是**不恰当、不严谨的**。
- **正确物理基石**：BB84 的安全性严格基于 **“非正交态不可区分定理”**。更精确的表述为：**区分非正交态必然会干扰至少其中一个态**。
